Avaliacao 360 - Guia de Configuracao

Sistema digital de avaliacao anonima para tecnicos de campo

1 Criar a planilha no Google

  1. Abra o Google Sheets: sheets.google.com
  2. Clique em "Em branco" para criar uma planilha nova
  3. Renomeie para "Avaliacao 360" (clique no nome "Planilha sem titulo" no canto superior esquerdo)
  4. Na primeira linha (linha 1), escreva esses cabecalhos, um em cada coluna:
Data Equipe Periodo Gestor+ Gestor- Empresa NotaEmp Sistema NotaSist Geral
Dica: Os cabecalhos completos sao: Data, Equipe, Periodo, GestorPositivos, GestorNegativos, EmpresaComentario, EmpresaNota, SistemaComentario, SistemaNota, SatisfacaoGeral

2 Criar o script que recebe as respostas

  1. Na planilha, clique no menu Extensoes > Apps Script
  2. Vai abrir uma tela com codigo. Apague tudo que tiver la
  3. Cole o codigo abaixo:
function doPost(e) { var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); var data = JSON.parse(e.postData.contents); sheet.appendRow([ new Date(), data.equipe, data.periodo, data.gestorPositivos, data.gestorNegativos, data.empresaComentario, data.empresaNota, data.sistemaComentario, data.sistemaNota, data.satisfacaoGeral ]); return ContentService .createTextOutput(JSON.stringify({status: 'success'})) .setMimeType(ContentService.MimeType.JSON); } function doGet(e) { var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); var data = sheet.getDataRange().getValues(); var headers = data[0]; var rows = data.slice(1); var result = rows.map(function(row) { var obj = {}; headers.forEach(function(header, i) { obj[header] = row[i]; }); return obj; }); return ContentService .createTextOutput(JSON.stringify(result)) .setMimeType(ContentService.MimeType.JSON); }
  1. Aperte Ctrl+S para salvar
  2. Coloque o nome do projeto como "Avaliacao 360"

3 Publicar o script

  1. Clique em Implantar > Nova implantacao (botao azul no canto superior direito)
  2. Clique na engrenagem ao lado de "Selecionar tipo" e escolha "App da Web"
  3. Preencha:
    • Descricao: Avaliacao 360
    • Executar como: Eu mesmo
    • Quem tem acesso: Qualquer pessoa
  4. Clique em Implantar
  5. Vai pedir autorizacao. Clique Autorizar acesso > escolha sua conta Google
  6. Se aparecer "Google nao verificou esse app", clique em Avancado > "Acessar Avaliacao 360 (nao seguro)" - pode confiar, e seu proprio script
  7. COPIE O LINK que aparece na tela (URL da implantacao)
O link sera parecido com:
https://script.google.com/macros/s/AKfycbx.../exec

4 Conectar no sistema

  1. Abra: rodrigolsilvao.github.io/avaliacao-360/setup.html
  2. Cole o link copiado no passo 3 no campo "URL do Google Apps Script"
  3. Clique em "Testar Conexao"
  4. Se aparecer verde, clique "Salvar"
  5. Pronto! O sistema esta conectado
