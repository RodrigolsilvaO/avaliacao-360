Modo Demo (localStorage)
Os dados ficam salvos apenas no navegador

No modo demo, todas as avaliações são armazenadas localmente no navegador. Isso é útil para testes, mas os dados não são compartilhados entre dispositivos e podem ser perdidos ao limpar os dados do navegador.

Integração com Google Sheets
Passo a passo para conectar ao Google Sheets
1
Crie uma Planilha no Google Sheets
Acesse sheets.google.com e crie uma nova planilha. Na primeira linha, adicione os seguintes cabeçalhos:
Data | Equipe | Período | Gestor - Pontos Positivos | Gestor - Pontos Negativos | Empresa - Comentário | Empresa - Nota | Sistema - Comentário | Sistema - Nota | Satisfação Geral

Dica: Use o separador "|" como guia. Cada item deve estar em uma coluna separada (A até J).

2
Abra o Editor de Scripts
Na planilha, vá em Extensões > Apps Script. Isso abrirá o editor de código do Google. Apague todo o conteúdo existente no editor.
3
Cole o Código do Script
Copie o código abaixo e cole no editor do Apps Script. Este código cria uma API que recebe e retorna dados da planilha.
// Avaliação 360 - Script para Google Sheets // Cole este código em: Extensões > Apps Script function doPost(e) { var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); var data = JSON.parse(e.postData.contents); sheet.appendRow([ new Date(), data.equipe, data.periodo, data.gestorPositivos, data.gestorNegativos, data.empresaComentario, data.empresaNota, data.sistemaComentario, data.sistemaNota, data.satisfacaoGeral ]); return ContentService .createTextOutput(JSON.stringify({status: 'success'})) .setMimeType(ContentService.MimeType.JSON); } function doGet(e) { var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); var data = sheet.getDataRange().getValues(); var headers = data[0]; var rows = data.slice(1); var result = rows.map(function(row) { var obj = {}; headers.forEach(function(header, i) { obj[header] = row[i]; }); return obj; }); return ContentService .createTextOutput(JSON.stringify(result)) .setMimeType(ContentService.MimeType.JSON); }
4
Implante como Aplicativo Web
No editor do Apps Script:
  1. Clique em Implantar > Nova implantação
  2. Em "Tipo", selecione App da Web
  3. Em "Executar como", selecione Eu mesmo
  4. Em "Quem pode acessar", selecione Qualquer pessoa
  5. Clique em Implantar
  6. Autorize o acesso quando solicitado
  7. Copie a URL do aplicativo web gerada

Observações Importantes
  • A planilha do Google Sheets serve como banco de dados gratuito e acessível
  • Mesmo no modo Google Sheets, os dados também são salvos localmente como backup
  • O sistema não coleta nenhum dado pessoal dos técnicos
  • Para alterar a senha do dashboard, edite o código em config.js
  • Recomendamos compartilhar a planilha apenas com coordenadores e gestores autorizados
  • Se a integração falhar, os dados continuam seguros no armazenamento local
